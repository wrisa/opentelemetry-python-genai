Use `call_instrumentation_lib_and_save_telemetry.py` script to compare logs / spans generated by 2 different versions of this instrumentation (one from your local branch vs the one from main for example). This folder is not included in the PyPi package release.

1. Update `pyproject.toml` to point to the branch you want to test. Run uv sync to install the dependencies in a venv.
2. Then run `uv run --env-file=main.env call_instrumentation_lib_and_save_telemetry.py` after updating the variables in `main.env`. The script will create a `test_span.textproto` and `test_log.textproto` in the current working directory that the instrumentation generated.
3. Then you can update `pyproject.toml` to point to another branch and compare the generated
test_span.textproto and test_log.textproto with the ones generated when running this script with the new branch.
The test_span.textproto and test_log.textproto files will be overwritten each time the script is run unless you update the filename in the script.
